# Dine-Log

Dine-Log is a web application for managing restaurant information, including details such as name, location, area, description, and average cost. It is built using Next.js, Mongoose, and MongoDB.

## Usage 
1. Clone the repository 
```bash 
git clone git@github.com:Git-Cat-21/DineLog.git
cd DineLog
```

2. Install Dependencies
```bash
npm install
```

3. Create a `.env` file in the root directory and add the following:
```bash
MongoURL="mongodb://localhost:27017/restaurants"
```
4. Start the developement server:
```bash
npm run dev
```

5. Open the app in your browser at this [link](https://localhost:3000).
